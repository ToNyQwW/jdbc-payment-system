# Система Платежей (JDBC)

## Описание

Проект демонстрирует работу с JDBC и CRUD операциями. Клиенты имеют счета и кредитные карты, могут оплачивать заказы, переводить деньги между счетами, блокировать кредитные карты и аннулировать счета.

## Сущности

* **Клиент (Client)**: содержит информацию о клиенте, включая имя, фамилию и контактные данные.
* **Счёт (Account)**: представляет банковский счёт клиента, включая баланс и статус.
* **Кредитная Карта (CreditCard)**: представляет кредитную карту клиента, включая номер, лимит и статус.
* **Заказ (Order)**: представляет заказ, включая дату, сумму и статус оплаты.
* **Платёж (Payment)**: представляет платёж, включая сумму, дату и связанные счета.
